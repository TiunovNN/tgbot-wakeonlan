# tgbot-wakeonlan
Telegram Bot to wake on lan computers

# installation
```shell
# Linux and MacOS
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# Windows
c:\Python38\python -m venv c:\path\to\myenv
.\env\Scripts\activate
pip install -r requirements.txt
```

# settings

All settings are located in the file `settings.py`

# run
```shell
python main.py
```
